Built a fully responsive YouTube Clone using Frontend Development technologies like HTML, CSS, and JavaScript. The project includes real-time page navigation between Home, Video Player, Subscriptions, Library, and Profile pages, giving a real app-like experience. Implemented video playback using embedded YouTube videos, interactive sidebar navigation, responsive layouts, search functionality, profile section, and modern UI design inspired by the original YouTube platform. This project helped in improving concepts of DOM manipulation, page routing, local storage usage, responsive web design, event handling, and frontend application structure.
